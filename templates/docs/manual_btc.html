BTC交易分析 - 使用手册

操作步骤、结果解释、API说明与案例演示

1. 操作步骤

  1. 输入待分析的比特币地址或交易哈希。
  2. 选择分析模式：地址分析、交易追踪或UTXO分析。
  3. 点击"开始分析"，系统将通过Blockstream API获取区块链数据。
  4. 查看分析结果，包括交易历史、余额变化、输入输出详情。
  5. 选择导出格式（JSON/CSV），保存分析报告。

2. 结果解释

地址信息

  • 地址余额：当前未花费的BTC数量
  • 总接收金额：该地址历史接收的BTC总量
  • 总发送金额：该地址历史发送的BTC总量
  • 交易数量：该地址参与的交易总数
  • 首次交易：地址首次出现的时间

交易详情

  • 交易ID：交易的唯一标识符
  • 输入：资金的来源地址和金额
  • 输出：资金的目标地址和金额
  • 手续费：矿工费用
  • 确认数：交易被确认的区块数

UTXO分析

  • UTXO列表：未花费的交易输出
  • UTXO数量：当前可花费的输出数量
  • UTXO总价值：所有UTXO的BTC总和

3. API密钥说明

本工具使用Blockstream API，无需API密钥。

Blockstream提供公开的比特币区块链数据查询接口，用户可直接使用本工具进行交易分析，无需注册或获取API密钥。

API文档：https://blockstream.info/api

提示：Blockstream API有请求频率限制，如需大量分析建议间隔执行。

4. 案例演示

案例背景

某网络赌博案件，嫌疑人使用比特币收取赌资，需要分析资金来源和去向。

操作过程

  1. 输入嫌疑人比特币地址：bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh
  2. 选择地址分析模式
  3. 查看交易历史：发现该地址接收多笔小额转账
  4. 分析输入来源：识别出多个打款地址
  5. 导出CSV报告，包含所有交易记录

结论

通过BTC交易分析，成功获取赌资来源地址列表，为后续调查提供证据支持。
